About project

This is a project for class JAVA-NATIONAL-05, from Scoala Informala de It, in wich present the overall knowledge of programming in Java. The tehnologies used are Spring, JPA, Postgresql for back-end, Thymeleaf, Bootstrap for front-end.
The project present a virtual airline portal, in wich we added some airports, airlines, and we can navigate through them, finding the designated flights.
We tried to cover as many knowledge we could in a designated period of time, impossed by the mentor, for testing purpose.

Hope you enjoyed the "flight", and we see you soon.
